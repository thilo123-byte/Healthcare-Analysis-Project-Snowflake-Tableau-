## Heathcare Analytics Project(SQL + Tableau)

# Project Overview
Tnis project is a simulated heathcare analytics solution developed using SQL,Snowflake and Tableau. The goal of the project was to demonstrate data analytics,reporting and visualization within a Healthcare Operations environment. The project focuses on analyzing patient appointments,operatonal performance and heathcare insights through SQL anlytics and interactive dashboards.

# Objectives
The main objectives of this project were to;
- Build a healthcare anaytics workflow.
- Practise SQL quirrying and transformation techniques.
- Practise developing a cloud-based data warehouse using Snowfake,including data loading,pre-processing in staging,building schemas and      exporting results.
- Creating interactive dshboards and story teilling through Tableau 
- Simulate operational heathcare reporting and KPI tracking

# Tools and Technologies
- Snowflake : Cloud Data Warehouse
- SQL : Data cleaning,Transformation and Analysis
- Tableau - Dashboarding,reporting and story telling
- Github : Portfoli hosting and management
- Excel/CSV : Source Data

# Dataset
The simulated dataset includes;
- 1000 rows-table of Patient demographics ie ID,Name,Age,Sex,City,etc
- 1000 rows-table of patient appointment status plus doctor and date
- Both data sets mimic typical healthcare datasets with missing values,duplicates,cancellations,pending status,no appoinment dates,etc

# Project work flow
- Created database,schema ie raw and clean schema,stage,staging table and other relevant tables
- Loaded,staged data tables in Snowflake using SQL quiries plus Snowsight UI
- Handled missing values,duplicates,inconsistant data formats like dates,names,emails,etc
- Performed joins,aggregated operations and transformations
- Exported results,clean tables,flagged nulls,etc in CSV format to Tablea
- Created sheets,calculated fields,charts,dashboards and stories in Tableau
- Shared in Tableau public

# Tableau features
- build-up sheets containing appointments by city,doctor workload analysis,time-trend analysis,appointment status analysis,etc
- interactive operations dashboard
- 3 page story with operations dashboard,pending appointments tracking and final page with insights

# Business questions Explored
